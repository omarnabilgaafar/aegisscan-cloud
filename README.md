# 🛡️ AegisScan Cloud

> Product-grade Web Vulnerability Scanner with SaaS UI & Executive PDF Reporting

---

## 🚀 Overview

**AegisScan Cloud** is a full-stack cybersecurity platform that simulates real-world vulnerability scanning workflows and presents them through a modern SaaS-style interface.

It is designed to **look and feel like a real security product**, not just a script — making it ideal for:

* 💼 Portfolio projects
* 🎯 Internship / job applications
* 🧠 Demonstrating security + full-stack skills

---

## ✨ Key Highlights

* ⚡ Guided Demo Scanning Experience
* 🧠 Realistic vulnerability simulation (XSS, SQLi, CSRF, Headers)
* 📊 Threat Monitoring Dashboard
* 📄 Executive-level PDF Reports
* 🎯 Prebuilt demo targets
* 🌑 Dark SaaS UI (Cyber Theme)
* 🧪 Demo Mode (Unlimited scans)

## ⚙️ Run Locally

```bash
git clone https://github.com/your-username/aegisscan-cloud.git
cd smart-web-vulnerability-scanner

docker compose up --build
```

Open in browser:

```
http://127.0.0.1:5000
```
