# CV Pitch

**AegisScan Cloud** is a SaaS-style web vulnerability scanning platform built for educational and authorized defensive-security workflows. It combines scan execution, severity analytics, PDF reporting, workspace-based onboarding, plan-aware dashboards, and an admin view in one product-style Flask application.

## Resume-ready points
- Built a Flask security platform with workspace onboarding, role-aware access, and admin analytics.
- Designed a dashboard that aggregates risk score, severity counts, recent findings, and workspace scan usage.
- Implemented PDF export for executive-ready vulnerability reports.
- Added SQLite schema migration logic for organizations, plans, API keys, and workspace-linked scan records.
- Crafted a polished dark SaaS UI to present the project as a product rather than a basic CRUD demo.
